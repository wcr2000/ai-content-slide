# Part 06 — ไม่รู้จะพูดอะไร? ให้ AI เขียนบทพูดให้

> เวลา: 10-15 นาที

## เป้าหมาย

ให้ผู้เรียนสามารถใช้ AI สร้างบทพูด (Script) สำหรับถ่ายคลิปลง Social Media ได้ทุกแพลตฟอร์ม ไม่ว่าจะเป็น TikTok, IG Reels, YouTube Shorts, หรือ Facebook Reels

## ปัญหาที่เจอบ่อย

- เปิดกล้องแล้ว **ไม่รู้จะพูดอะไร**
- พูดวนไปวนมา ไม่มีจุดเริ่ม-จบ
- คลิปยาวเกินไป คนดูปิดก่อน
- อยากทำคลิปให้ความรู้ แต่จัดลำดับไม่ถูก

**AI ช่วยได้ทั้งหมด** — แค่บอกหัวข้อ AI เขียนบทพูดให้ เราแค่อ่านแล้วปรับเป็นภาษาตัวเอง แล้วเปิดกล้องถ่ายได้เลย

## โครงสร้างคลิปที่ดี (ใช้ได้ทุกแพลตฟอร์ม)

```
Hook (1-3 วินาที)    → ดึงคนดูให้หยุดเลื่อน
เนื้อหา (15-50 วินาที) → สาระหลักที่อยากบอก
CTA (3-5 วินาที)     → ชวนให้ทำอะไรสักอย่าง (follow/save/comment)
```

### ทำไม Hook สำคัญมาก?

- TikTok/Reels: คนตัดสินใจดูต่อหรือปัดผ่านภายใน **1-3 วินาทีแรก**
- ถ้า Hook ไม่ดี เนื้อหาดีแค่ไหนก็ไม่มีคนดู

## ตัวอย่าง: ให้ AI เขียน Script คลิปสั้น

### ตัวอย่างที่ 1 — คลิปให้ความรู้ (30 วินาที)

**Prompt:**
> เขียนบทพูดสำหรับคลิปสั้น 30 วินาที หัวข้อ "3 สิ่งที่ทำตอนเช้าแล้วผิวดี" โทนกันเอง เหมือนคุยกับเพื่อน มี hook ดึงดูด

**ตัวอย่าง Output:**

> **[Hook]** "ทำ 3 สิ่งนี้ทุกเช้า ผิวเปลี่ยนภายใน 2 สัปดาห์"
>
> **[เนื้อหา]**
> "อย่างแรก — ตื่นมาดื่มน้ำอุ่น 1 แก้วก่อนเลย ช่วยล้างสารพิษ
>
> อย่างที่สอง — ทาครีมกันแดดทุกวัน แม้อยู่บ้าน แสงจากหน้าจอก็ทำร้ายผิวนะ
>
> อย่างที่สาม — กินผักผลไม้สีส้มสีแดง เช่น มะเขือเทศ แครอท วิตามินเอเยอะ ผิวจะโกลว์ขึ้นจริงๆ"
>
> **[CTA]** "ลองทำดูนะ แล้วกลับมาบอกผลใน comment! กด follow ไว้ จะมีทิปส์แบบนี้ทุกวัน"

### ตัวอย่างที่ 2 — คลิปขายของ (30 วินาที)

**Prompt:**
> เขียนบทพูดคลิปสั้น 30 วินาที ขายครีมบำรุงผิวหน้า โทนน่าเชื่อถือแต่ไม่เป็นทางการ เน้นบอกผลลัพธ์จริง

**ตัวอย่าง Output:**

> **[Hook]** "ครีมตัวนี้ ใช้มา 2 อาทิตย์ ผิวเปลี่ยนไปขนาดนี้เลย"
>
> **[เนื้อหา]**
> "ก่อนหน้านี้ผิวหมองมาก สิวก็ขึ้น พอลองใช้ตัวนี้ อาทิตย์แรกยังไม่เห็นผลนะ แต่พอเข้าอาทิตย์ที่สอง ผิวสว่างขึ้นชัดเลย
>
> เนื้อครีมบางเบา ซึมเร็ว ไม่เหนียว ทาตอนเช้าก่อนแต่งหน้าได้เลย
>
> ตัวนี้มี Niacinamide กับ Vitamin C ช่วยเรื่องผิวกระจ่างใสโดยเฉพาะ"
>
> **[CTA]** "สนใจ DM มาเลยนะคะ ตอนนี้มีโปรส่งฟรี!"

### ตัวอย่างที่ 3 — คลิปเล่าเรื่อง/แชร์ประสบการณ์ (60 วินาที)

**Prompt:**
> เขียนบทพูดคลิป 60 วินาที เล่าประสบการณ์เปิดร้านออนไลน์ครั้งแรก โทนจริงใจ มี hook ดึงดูด เหมาะกับ TikTok

**ตัวอย่าง Output:**

> **[Hook]** "เดือนแรกที่เปิดร้าน ขายได้ 0 บาท"
>
> **[เนื้อหา]**
> "จริงๆ นะ เดือนแรกไม่มีใครซื้อเลย โพสต์ไปวันละ 3 รูป ไม่มีใครสนใจ
>
> ตอนนั้นท้อมาก คิดว่าจะเลิกดีไหม แต่สิ่งที่เปลี่ยนทุกอย่างคือ เริ่มถ่ายคลิปรีวิวสินค้าตัวเอง ไม่ได้ถ่ายสวยหรูนะ แค่ถ่ายจากมือถือ พูดตรงๆ ว่าสินค้าดียังไง
>
> คลิปแรกยอดวิวแค่ 200 แต่มีคนทักมาซื้อ 3 คน ตอนนั้นแหละที่รู้เลยว่า คนซื้อเพราะเชื่อเรา ไม่ใช่เพราะภาพสวย
>
> เดือนที่ 3 ยอดขายทะลุหมื่น เดือนที่ 6 ลาออกจากงานประจำเลย"
>
> **[CTA]** "ถ้ากำลังเริ่มต้นเหมือนกัน อย่าเพิ่งท้อนะ กด follow แล้วจะมาแชร์เรื่อยๆ ว่าทำยังไง"

## เทคนิค Hook ที่ได้ผล

| เทคนิค | ตัวอย่าง Hook | เหมาะกับ |
|--------|-------------|---------|
| **ตัวเลข** | "3 สิ่งที่คนผิวสวยทำทุกเช้า" | คลิปให้ความรู้ |
| **คำถาม** | "เคยสงสัยไหมว่า ทำไมหน้าใสไม่สม่ำเสมอ?" | คลิปแก้ปัญหา |
| **ความขัดแย้ง** | "ครีมราคา 100 บาท ดีกว่าครีมหลักพัน" | คลิปรีวิว |
| **ความลับ** | "สิ่งที่ร้านอาหารไม่อยากให้รู้" | คลิปเปิดโปง/เบื้องหลัง |
| **ก่อน-หลัง** | "จากยอด 0 เป็นหมื่นใน 3 เดือน" | คลิปแชร์ประสบการณ์ |
| **ท้าทาย** | "ลองทำแบบนี้ 7 วัน แล้วดูผล" | คลิป challenge |

**Prompt หา Hook:**
> สร้าง Hook สำหรับคลิปสั้น 10 แบบ หัวข้อ [xxx] ใช้เทคนิคหลากหลาย ทั้งคำถาม, ตัวเลข, ความขัดแย้ง, ความลับ

## ปรับ Script ตามแพลตฟอร์ม

| แพลตฟอร์ม | ความยาวแนะนำ | สไตล์ | จุดสำคัญ |
|-----------|-------------|-------|---------|
| **TikTok** | 15-60 วินาที | กันเอง, เร็ว, สนุก | Hook ต้องแรง, ตัดฉากบ่อย |
| **IG Reels** | 15-60 วินาที | สวยงาม, aesthetic | ภาพสวย + เนื้อหาดี |
| **YouTube Shorts** | 30-60 วินาที | ให้ความรู้, ลงลึกกว่า | เน้นสาระ, ชวน subscribe |
| **Facebook Reels** | 15-60 วินาที | กว้าง, เข้าถึงง่าย | ภาษาง่าย, ทุกวัย |
| **YouTube (ยาว)** | 5-15 นาที | ให้ความรู้จัดเต็ม | มี intro, เนื้อหาแบ่ง section, outro |

**Prompt ปรับข้ามแพลตฟอร์ม:**
> ปรับ script คลิปนี้ให้เหมาะกับ [TikTok/YouTube/Facebook]:
> "[วาง script เดิม]"
> ปรับความยาว โทน และ CTA ให้เข้ากับแพลตฟอร์ม

## สูตร Script สำหรับคลิปแต่ละแบบ

### คลิปให้ความรู้ (Education)

```
Hook: "X สิ่งที่คุณ [ทำผิด/ไม่รู้] เรื่อง [หัวข้อ]"
ข้อ 1: อธิบายสั้นๆ + เหตุผล
ข้อ 2: อธิบายสั้นๆ + เหตุผล
ข้อ 3: อธิบายสั้นๆ + เหตุผล
CTA: "กด save ไว้ แล้ว follow เพื่อดูทิปส์ใหม่ๆ"
```

### คลิปขายของ (Selling)

```
Hook: ผลลัพธ์ที่น่าสนใจ / ก่อน-หลัง
ปัญหา: บอกปัญหาที่คนดูมี (ให้รู้สึกว่า "ใช่เลย!")
สินค้า: แนะนำสินค้า + จุดเด่น 2-3 ข้อ
Social Proof: คนอื่นใช้แล้วเป็นยังไง
CTA: "DM มาเลย" / "กดลิงก์ใน bio"
```

### คลิปเล่าเรื่อง (Storytelling)

```
Hook: เหตุการณ์ที่น่าสนใจ / จุดพลิก
สถานการณ์: เกิดอะไรขึ้น
ปัญหา/อุปสรรค: เจออะไร
จุดเปลี่ยน: ทำยังไงถึงผ่านมาได้
บทเรียน: สิ่งที่ได้เรียนรู้
CTA: "ถ้าชอบเรื่องแบบนี้ กด follow ไว้"
```

### คลิปรีวิว (Review)

```
Hook: ผลลัพธ์สุดท้าย / ความประทับใจแรก
สิ่งที่ชอบ: 2-3 ข้อ
สิ่งที่ไม่ชอบ: 1 ข้อ (ทำให้น่าเชื่อถือ)
เหมาะกับใคร: บอกชัด
สรุป: แนะนำหรือไม่?
CTA: "comment บอกด้วยนะว่าอยากให้รีวิวอะไรต่อ"
```

## Tips

- **อย่าท่องบทพูด** — อ่านให้เข้าใจ แล้วพูดเป็นภาษาตัวเอง จะเป็นธรรมชาติกว่า
- **ถ่ายหลาย take** — ไม่ต้อง perfect ตั้งแต่ครั้งแรก
- **ตัดส่วนเงียบออก** — ให้คลิปกระชับ ไม่มีช่วงเว้นว่างนาน
- **ใส่ text overlay** — คนจำนวนมากดูคลิปแบบปิดเสียง ข้อความบนจอช่วยได้มาก
- **1 คลิป = 1 เรื่อง** — อย่ายัดเยอะเกินไป เลือกแค่ประเด็นเดียว
- Script ของ AI เป็นแค่ **โครงร่าง** ให้เราปรับเป็นภาษาของเราเอง

## ข้อควรระวัง

- อย่าพูดเร็วเกินไป แม้คลิปจะสั้น — พูดชัด ดีกว่าพูดเร็ว
- ตรวจสอบข้อมูลก่อนพูดในคลิป โดยเฉพาะเรื่องสุขภาพ, การเงิน
- ระวังเรื่องลิขสิทธิ์เพลงประกอบ — ใช้เพลงที่แพลตฟอร์มมีให้
- อย่าอ้างผลลัพธ์เกินจริง เช่น "ใช้แล้วหายป่วย" "รวยชัวร์"
