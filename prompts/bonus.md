# Prompts — Bonus: เทคนิคขั้นสูง & Prompt สำเร็จรูป

## Prompt 1: สร้าง Brand Prompt (ใช้ซ้ำได้ทุกครั้ง)

```
จากนี้ไป ทุกครั้งที่เขียน caption IG ให้ใช้แนวทางนี้:
- แบรนด์: [ชื่อแบรนด์/ธุรกิจ]
- ขาย/ให้บริการ: [สินค้า/บริการ]
- กลุ่มเป้าหมาย: [เพศ, อายุ, ไลฟ์สไตล์]
- โทนเสียง: [กันเอง/มืออาชีพ/น่ารัก]
- ใช้ emoji: [เยอะ/พอประมาณ/น้อย]
- ลงท้ายด้วย: CTA เสมอ
- ภาษา: ไทย

ยืนยันว่าเข้าใจแล้วตอบ "พร้อมแล้วครับ/ค่ะ"
```

> **วิธีใช้:** วาง prompt นี้ตอนเริ่มแชทใหม่ทุกครั้ง แล้ว AI จะจำสไตล์ของเราตลอดบทสนทนา

## Prompt 2: วิเคราะห์โพสต์ที่ Engage ดี

```
วิเคราะห์ caption IG นี้ว่าทำไมถึง engage ดี:

"[วาง caption ที่ได้ engagement สูง]"

บอกมา 5 ข้อว่าอะไรทำให้ดี แล้วช่วยเขียน caption ใหม่ในสไตล์เดียวกัน สำหรับหัวข้อ [หัวข้อใหม่]
```

## Prompt 3: วิเคราะห์คู่แข่ง

```
ฉันมีร้าน [ประเภท] คู่แข่งในตลาดมักโพสต์เนื้อหาแบบนี้: [อธิบายสไตล์คู่แข่ง]

ช่วยแนะนำ 5 แนวทางทำคอนเทนต์ IG ที่แตกต่าง โดดเด่นกว่าคู่แข่ง พร้อมตัวอย่าง caption
```

## Prompt 4: สร้าง Hook สำหรับ Reels

```
สร้าง hook (ประโยคเปิด) สำหรับ Reels IG 10 แบบ หัวข้อ [หัวข้อ]
- แต่ละ hook ต้องดึงดูดภายใน 3 วินาทีแรก
- ใช้เทคนิค: คำถาม, ความขัดแย้ง, ตัวเลข, ความลับ
- ระบุว่า hook แต่ละแบบใช้เทคนิคอะไร
```

**ตัวอย่าง Output:**

> 1. "คุณกำลังทำ 3 สิ่งนี้ผิดเกี่ยวกับสุขภาพ!" (ความขัดแย้ง)
> 2. "อาหารเช้าแบบไหนที่ทำให้อ้วนโดยไม่รู้ตัว?" (คำถาม)
> 3. "ฉันเปลี่ยนนิสัยแค่ 1 อย่าง น้ำหนักลด 5 กก." (ตัวเลข + เรื่องเล่า)
> 4. "สิ่งที่หมอไม่ค่อยบอก เรื่องการนอนหลับ" (ความลับ)
> ...

## Prompt 5: เขียน Script Reels สั้น

```
เขียน script สำหรับ Reels IG ความยาว 30-60 วินาที หัวข้อ "[หัวข้อ]"
โครงสร้าง:
- Hook (3 วินาที): ดึงดูดคนดู
- เนื้อหา (20-40 วินาที): สาระหลัก
- CTA (5 วินาที): ชวน follow/save/share
ระบุ: คำพูด, action, ข้อความบนจอ (text overlay)
```

## Prompt 6: Batch สร้างคอนเทนต์ทั้งสัปดาห์

```
สร้างคอนเทนต์ IG ครบชุดสำหรับ 1 สัปดาห์ ธุรกิจ [ประเภท]:

สำหรับแต่ละวัน ให้:
1. หัวข้อ + รูปแบบโพสต์
2. Caption (80-120 คำ, มี emoji, CTA)
3. Hashtag 15 อัน
4. Prompt สำหรับสร้างภาพ (ใช้กับ ChatGPT/Canva AI)
5. เวลาโพสต์ที่แนะนำ
```

## Prompt 7: A/B Test Caption

```
เขียน caption IG หัวข้อ "[หัวข้อ]" 2 เวอร์ชัน สำหรับ A/B test:

เวอร์ชัน A: เน้น [emotional/เล่าเรื่อง]
เวอร์ชัน B: เน้น [ข้อมูล/ตรงประเด็น]

แต่ละเวอร์ชันใส่ emoji, CTA, hashtag
อธิบายว่าเวอร์ชันไหนน่าจะเหมาะกับกลุ่มเป้าหมายไหน
```

## Prompt 8: แปลง Blog → IG Post

```
แปลงบทความนี้เป็นโพสต์ IG:

"[วางบทความหรือเนื้อหายาว]"

แปลงเป็น:
1. Carousel 5-7 ภาพ (เขียน text แต่ละภาพ)
2. Caption ประกอบ (100-150 คำ)
3. Hashtag 15 อัน
4. Reels script สั้น 30 วินาที
```
