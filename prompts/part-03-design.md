# Prompts — Part 03: สร้างภาพ / ดีไซน์ด้วย AI

## Prompt 1: สร้างภาพโปสเตอร์

```
สร้างภาพโปสเตอร์สำหรับ Instagram Post เรื่อง [หัวข้อ]
- สไตล์: [minimal/modern/vintage/cute]
- สี: [โทนสี]
- มีองค์ประกอบ: [ระบุ]
- ขนาด: สี่เหลี่ยมจัตุรัส (1:1)
```

**ตัวอย่าง:**
```
สร้างภาพโปสเตอร์สำหรับ Instagram Post เรื่องสุขภาพ
- สไตล์: minimal, flat design
- สี: พาสเทล เขียว-ชมพู
- มีองค์ประกอบ: ผักผลไม้, ไอคอนหัวใจ, ไอคอนออกกำลังกาย
- ขนาด: สี่เหลี่ยมจัตุรัส (1:1)
- พื้นหลังสีครีม
```

## Prompt 2: สร้างภาพสินค้า / Mockup

```
สร้างภาพถ่ายสินค้า [ประเภทสินค้า]
- จัดวางแบบ: [flat lay/ถือในมือ/วางบนโต๊ะ]
- แสง: [แสงธรรมชาติ/studio/golden hour]
- พื้นหลัง: [สีพื้น/ธรรมชาติ/ร้าน]
- บรรยากาศ: [minimal/luxury/cozy]
```

**ตัวอย่าง:**
```
สร้างภาพถ่ายครีมบำรุงผิวบนโต๊ะหินอ่อนสีขาว
- จัดวางแบบ flat lay
- มีดอกไม้สดประกอบ
- แสงธรรมชาติยามเช้า นุ่มนวล
- บรรยากาศ luxury minimal
```

## Prompt 5: สร้าง Illustration ประกอบโพสต์

```
สร้างภาพ illustration สไตล์ [flat/hand-drawn/3D] เกี่ยวกับ [หัวข้อ]
- ตัวการ์ตูน: [มี/ไม่มี]
- สี: [โทนสี]
- ใช้สำหรับ: โพสต์ IG ให้ความรู้เรื่อง [xxx]
```

**ตัวอย่าง:**
```
สร้างภาพ illustration สไตล์ flat design เกี่ยวกับคนนั่งทำงานที่โต๊ะ
- มีตัวการ์ตูนผู้หญิงนั่งทำงานกับคอมพิวเตอร์
- มีต้นไม้และแก้วกาแฟบนโต๊ะ
- สีพาสเทล เขียว-ส้ม-ฟ้า
- พื้นหลังสีขาว
```

## สูตรรวม: องค์ประกอบ Prompt สร้างภาพ

ผสมจากรายการนี้ตามต้องการ:

| หมวด | ตัวเลือก |
|------|---------|
| สไตล์ | photo, illustration, flat design, watercolor, 3D, cartoon, minimal |
| สี | พาสเทล, warm tone, cool tone, monochrome, gradient, neon |
| แสง | natural light, golden hour, studio, soft, dramatic, neon glow |
| มุมกล้อง | top view, close-up, wide shot, flat lay, eye level |
| บรรยากาศ | cozy, luxury, playful, professional, aesthetic, vintage |
| ขนาด IG | Post (1:1), Story/Reels (9:16), Landscape (16:9) |
