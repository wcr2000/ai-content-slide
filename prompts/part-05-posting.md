# Prompts — Part 05: โพสต์ลง IG อย่างถูกต้อง

## Prompt 1: สร้าง Hashtag Set

```
แนะนำ hashtag สำหรับโพสต์ IG เรื่อง [หัวข้อ] สำหรับ [ประเภทธุรกิจ]
แบ่งเป็น 3 กลุ่ม:
- Popular (1M+ posts): 5 อัน
- Medium (100K-1M posts): 10 อัน
- Niche (ต่ำกว่า 100K posts): 10 อัน
รวม 25 อัน ผสมภาษาไทยและอังกฤษ
```

**ตัวอย่าง Output:**

> **Popular:** #สุขภาพดี #HealthyFood #กินคลีน #Healthy #อาหารสุขภาพ
> **Medium:** #อาหารคลีน #เมนูสุขภาพ #กินดีอยู่ดี #CleanEating #HealthyMeal #อาหารเช้าสุขภาพ #สูตรอาหารคลีน #HealthyRecipe #กินสะอาด #สุขภาพดีง่ายๆ
> **Niche:** #อาหารคลีนทำเอง #เมนูคลีนง่ายๆ #สุขภาพดีทุกวัน #กินคลีนง่ายๆ #อาหารเช้าทำง่าย #เมนูสุขภาพวัยทำงาน #คลีนไทย #สูตรอาหารสุขภาพง่ายๆ #อาหารเช้าคลีน #กินดีสุขภาพดี

## Prompt 2: Hashtag Bank (สร้างคลังสลับใช้)

```
สร้างคลัง hashtag 50 อัน สำหรับ [ประเภทธุรกิจ/หัวข้อ]
จัดกลุ่มเป็น 5 ชุด ชุดละ 10 อัน เพื่อสลับใช้แต่ละโพสต์
ผสม popular, medium, niche ในแต่ละชุด
```

## Prompt 3: เวลาโพสต์ที่เหมาะสม

```
แนะนำเวลาโพสต์ IG ที่ดีที่สุดสำหรับแต่ละวันในสัปดาห์
- ประเภทธุรกิจ: [ระบุ]
- กลุ่มเป้าหมาย: [เพศ, อายุ, อาชีพ]
- ประเทศ: ไทย
จัดเป็นตาราง ระบุเหตุผล
```

**ตัวอย่าง:**
```
แนะนำเวลาโพสต์ IG ที่ดีที่สุดสำหรับแต่ละวันในสัปดาห์
- ประเภทธุรกิจ: ร้านขายเสื้อผ้าออนไลน์
- กลุ่มเป้าหมาย: ผู้หญิง อายุ 25-35 ปี พนักงานออฟฟิศ
- ประเทศ: ไทย
จัดเป็นตาราง ระบุเหตุผล
```

## Prompt 4: เขียน Bio IG

```
เขียน Bio IG สำหรับ [ประเภทธุรกิจ/บุคคล]
- จุดเด่น: [ระบุ 2-3 ข้อ]
- CTA: [DM/ลิงก์/โทร]
- ใส่ emoji ที่เหมาะสม
- ไม่เกิน 150 ตัวอักษร
เขียนมา 3 แบบให้เลือก
```

**ตัวอย่าง:**
```
เขียน Bio IG สำหรับร้านขายเครื่องสำอางออนไลน์
- จุดเด่น: ของแท้ 100%, ส่งฟรี, มีรีวิวจริง
- CTA: DM สั่งซื้อ
- ใส่ emoji ที่เหมาะสม
- ไม่เกิน 150 ตัวอักษร
เขียนมา 3 แบบให้เลือก
```

**Output ที่คาดหวัง:**

> **แบบ 1:**
> 💄 เครื่องสำอางแท้ 100% | ส่งฟรีทั่วไทย
> ⭐ รีวิวจริงจากลูกค้า 1,000+
> 📦 สั่งวันนี้ ส่งพรุ่งนี้
> 👇 DM สั่งซื้อเลย!
>
> **แบบ 2:**
> ✨ Your Beauty Destination
> 💯 ของแท้ การันตี | ส่งฟรี
> 💬 รีวิวจริง ไม่จ้าง
> 📩 DM มาเลยค่ะ!
>
> **แบบ 3:**
> 🛍️ ช้อปเครื่องสำอางแท้ ราคาดี
> ✅ แท้ 100% ✅ ส่งฟรี ✅ รีวิวจริง
> 💌 DM สั่งซื้อ | ตอบไว 24 ชม.

## Prompt 5: ตั้งชื่อ Highlight

```
แนะนำชื่อ Highlight IG สำหรับ [ประเภทธุรกิจ] จำนวน [6-8] หมวด
- ใช้ emoji นำหน้า
- ชื่อสั้นกระชับ (1-2 คำ)
- ครอบคลุม: สินค้า, รีวิว, โปรโมชั่น, FAQ, เกี่ยวกับเรา, วิธีสั่งซื้อ
```

## Prompt 6: เขียน Alt Text สำหรับ Accessibility

```
เขียน alt text สำหรับรูปโพสต์ IG นี้:
- ภาพเป็น: [อธิบายสั้นๆ ว่าภาพเป็นอะไร]
- ใช้สำหรับ: โพสต์เรื่อง [หัวข้อ]
- ความยาวไม่เกิน 125 ตัวอักษร
- อธิบายให้คนที่มองไม่เห็นเข้าใจ
```
