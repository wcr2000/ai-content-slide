# Prompts — Part 02: เขียน Caption IG ด้วย AI

## Prompt 1: Caption พื้นฐาน

```
เขียน caption IG เรื่อง [หัวข้อ] สำหรับ [กลุ่มเป้าหมาย] โทน [น่ารัก/มืออาชีพ/กันเอง] ใส่ emoji ความยาว 150 คำ มี CTA ตอนท้าย
```

**ตัวอย่าง:**
```
เขียน caption IG เรื่องครีมกันแดด สำหรับสาววัยทำงาน โทนน่ารัก ใส่ emoji ความยาว 150 คำ มี CTA ชวน DM สั่งซื้อ
```

**Output ที่คาดหวัง:**
> ☀️ แดดเมืองไทยไม่เคยปราณีใคร~
> แต่ผิวเราต้องได้รับการปกป้องนะคะ 💕
>
> [ชื่อสินค้า] SPF50+ PA++++ เนื้อบางเบา ทาแล้วไม่วอก ไม่เหนียว ลงรองพื้นต่อได้เลย ✨
>
> สาวออฟฟิศต้องมี! ทาเช้าคุมได้ทั้งวัน 🏢
>
> 💬 DM มาเลยนะคะ มีโปรพิเศษสำหรับ follower!

## Prompt 3: Caption Carousel (ให้ความรู้)

```
เขียน caption สำหรับ Instagram Carousel 5 ภาพ หัวข้อ "[หัวข้อ]"
- ภาพที่ 1: หน้าปก (ดึงดูดให้ปัด)
- ภาพที่ 2-4: เนื้อหาหลัก
- ภาพที่ 5: สรุป + CTA
เขียน caption ประกอบด้วย พร้อม hashtag 15 อัน
```

**ตัวอย่าง:**
```
เขียน caption สำหรับ Instagram Carousel 5 ภาพ หัวข้อ "5 อาหารเช้าทำง่าย สุขภาพดี"
- ภาพที่ 1: หน้าปก (ดึงดูดให้ปัด)
- ภาพที่ 2-4: เนื้อหาหลัก (แต่ละเมนู)
- ภาพที่ 5: สรุป + CTA ชวน save
เขียน caption ประกอบด้วย พร้อม hashtag 15 อัน
```

**Output ที่คาดหวัง:**
> **ภาพ 1 (ปก):** "5 อาหารเช้าทำง่ายใน 10 นาที! 🥣 (ปัดดูเลย →)"
> **ภาพ 2:** "1. ข้าวโอ๊ตผลไม้ — แช่ข้ามคืน ตื่นมากินได้เลย"
> **ภาพ 3:** "2. ไข่ต้ม + ขนมปังโฮลวีต — โปรตีนเต็มๆ"
> **ภาพ 4:** "3. สมูทตี้กล้วยโยเกิร์ต — ปั่น 2 นาที จบ!"
> **ภาพ 5:** "กด Save ไว้ทำพรุ่งนี้เช้า! 📌"
>
> Caption: เช้าๆ ไม่ต้องคิดเยอะ ทำง่าย อร่อย สุขภาพดี ✨ กด save ไว้เลยนะ!
> #อาหารเช้า #สุขภาพดี #กินคลีน ...

## Prompt 7: Batch Caption (เขียนทีเดียวหลาย caption)

```
เขียน caption IG 5 ชิ้น สำหรับร้าน [ประเภท] ในสัปดาห์นี้:
- วันจันทร์: แนะนำสินค้า
- วันอังคาร: tips/ความรู้
- วันพุธ: เบื้องหลัง
- วันพฤหัสบดี: โปรโมชั่น
- วันศุกร์: สร้างแรงบันดาลใจ

แต่ละ caption ใส่ emoji ความยาว 80-120 คำ มี hashtag ชุดละ 10 อัน
```
