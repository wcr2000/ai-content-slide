# Prompts — Part 06: ให้ AI เขียนบทพูดสำหรับคลิป

## Prompt 1: Script คลิปสั้น (30 วินาที)

```
เขียนบทพูดสำหรับคลิปสั้น 30 วินาที
- หัวข้อ: [หัวข้อ]
- แพลตฟอร์ม: [TikTok / IG Reels / YouTube Shorts / Facebook]
- โทน: [กันเอง / มืออาชีพ / สนุก / จริงใจ]
- กลุ่มเป้าหมาย: [ระบุ]
โครงสร้าง:
- Hook (3 วินาที): ดึงดูดคนดูให้หยุดเลื่อน
- เนื้อหา (20 วินาที): สาระหลัก
- CTA (5 วินาที): ชวน follow/save/share
ระบุ: คำพูด, action ที่ต้องทำ, ข้อความบนจอ (text overlay)
```

**ตัวอย่าง:**
```
เขียนบทพูดสำหรับคลิปสั้น 30 วินาที
- หัวข้อ: 3 เมนูอาหารเช้าทำง่าย
- แพลตฟอร์ม: TikTok
- โทน: กันเอง สนุก
- กลุ่มเป้าหมาย: คนวัยทำงาน
โครงสร้าง:
- Hook (3 วินาที): ดึงดูดคนดูให้หยุดเลื่อน
- เนื้อหา (20 วินาที): สาระหลัก
- CTA (5 วินาที): ชวน follow/save
ระบุ: คำพูด, action ที่ต้องทำ, ข้อความบนจอ (text overlay)
```

## Prompt 2: Script คลิปยาว (3-5 นาที สำหรับ YouTube)

```
เขียนบทพูดสำหรับคลิป YouTube ความยาว [3-5] นาที
- หัวข้อ: [หัวข้อ]
- โทน: [ระบุ]
- กลุ่มเป้าหมาย: [ระบุ]
โครงสร้าง:
- Intro + Hook (15-30 วินาที): แนะนำตัว + ทำไมต้องดูคลิปนี้
- เนื้อหา (2-4 นาที): แบ่งเป็น section ชัดเจน
- Outro (15-30 วินาที): สรุป + CTA (subscribe/like/comment)
เขียนแบบพูดเป็นธรรมชาติ ไม่เหมือนอ่านบทความ
```

## Prompt 3: Script คลิปขายของ

```
เขียนบทพูดคลิปขายสินค้า ความยาว [30-60] วินาที
- สินค้า: [ชื่อ/ประเภทสินค้า]
- จุดเด่น: [ระบุ 3 ข้อ]
- ราคา/โปรโมชั่น: [ระบุ]
- แพลตฟอร์ม: [TikTok / IG / Facebook]
โครงสร้าง:
- Hook: ผลลัพธ์หรือปัญหาที่คนดูมี
- บอกปัญหา → แนะนำสินค้า → จุดเด่น → social proof
- CTA: วิธีสั่งซื้อ
โทน: จริงใจ น่าเชื่อถือ ไม่ขายของแรงเกินไป
```

**ตัวอย่าง:**
```
เขียนบทพูดคลิปขายสินค้า ความยาว 30 วินาที
- สินค้า: เซรั่มวิตามินซี
- จุดเด่น: ผิวกระจ่างใสใน 2 สัปดาห์, เนื้อบางเบาซึมไว, มี Niacinamide
- ราคา/โปรโมชั่น: 590 บาท ซื้อ 2 ลด 100
- แพลตฟอร์ม: TikTok
โครงสร้าง:
- Hook: ผลลัพธ์หรือปัญหาที่คนดูมี
- บอกปัญหา → แนะนำสินค้า → จุดเด่น → social proof
- CTA: DM สั่งซื้อ
โทน: จริงใจ น่าเชื่อถือ
```

## Prompt 4: Script คลิปเล่าเรื่อง / แชร์ประสบการณ์

```
เขียนบทพูดคลิปเล่าเรื่อง ความยาว [30-60] วินาที
- เรื่องที่จะเล่า: [สรุปสั้นๆ ว่าอยากเล่าเรื่องอะไร]
- บทเรียน/ข้อคิด: [สิ่งที่อยากให้คนดูได้]
- โทน: [จริงใจ / ตลก / สร้างแรงบันดาลใจ]
- แพลตฟอร์ม: [ระบุ]
โครงสร้าง: Hook (เหตุการณ์น่าสนใจ) → เล่าเรื่อง → จุดพลิก → บทเรียน → CTA
```

**ตัวอย่าง:**
```
เขียนบทพูดคลิปเล่าเรื่อง ความยาว 60 วินาที
- เรื่องที่จะเล่า: เปิดร้านออนไลน์ครั้งแรก เดือนแรกขายได้ 0 บาท แต่ไม่ยอมแพ้ จนเดือนที่ 6 ยอดทะลุแสน
- บทเรียน: ความสม่ำเสมอสำคัญกว่าความสมบูรณ์แบบ
- โทน: จริงใจ สร้างแรงบันดาลใจ
- แพลตฟอร์ม: TikTok
โครงสร้าง: Hook → เล่าเรื่อง → จุดพลิก → บทเรียน → CTA
```

## Prompt 5: Script คลิปรีวิว

```

- สิ่งที่รีวิว: [สินค้า/ร้านอาหาร/สถานที่/แอป]

- สิ่งที่ไม่ชอบ: [ระบุ 1 ข้อ เพื่อความน่าเชื่อถือ]
- ให้คะแนน: [x/10]
- แพลตฟอร์ม: [ระบุ]
- โทน: ตรงไปตรงมา จริงใจ
ระบุ: คำพูด, action, text overlay
```

## Prompt 6: สร้าง Hook 10 แบบ

```
สร้าง Hook (ประโยคเปิดคลิป) 10 แบบ สำหรับหัวข้อ "[หัวข้อ]"
- แต่ละ Hook ต้องดึงดูดภายใน 3 วินาทีแรก
- ใช้เทคนิคหลากหลาย: คำถาม, ตัวเลข, ความขัดแย้ง, ความลับ, ก่อน-หลัง, ท้าทาย
- ระบุว่าแต่ละ Hook ใช้เทคนิคอะไร
- เหมาะกับ [TikTok / IG Reels / YouTube Shorts]
```

## Prompt 7: แตก 1 หัวข้อ → หลายคลิป

```
จากหัวข้อ "[หัวข้อ]" ช่วยแตกเป็นไอเดียคลิปสั้น 7 คลิป
แต่ละคลิปให้:
1. Hook (ประโยคเปิด)
2. เนื้อหาสั้นๆ 2-3 ประโยค
3. CTA
4. แพลตฟอร์มที่เหมาะ
5. ความยาวที่แนะนำ
```

## Prompt 9: Script คลิป How-to / สอนทำ

```
เขียนบทพูดคลิปสอนทำ [สิ่งที่จะสอน] ความยาว [30-60] วินาที
- จำนวนขั้นตอน: [3-5] ขั้นตอน
- กลุ่มเป้าหมาย: [ระบุ]
- แพลตฟอร์ม: [ระบุ]
โครงสร้าง:
- Hook: บอกผลลัพธ์ที่จะได้
- Step 1-3: อธิบายสั้นๆ แต่ละขั้นตอน
- CTA: ชวน save ไว้ทำตาม
ระบุ: คำพูด, action/สิ่งที่ถ่าย, text overlay
```

## Prompt 10: Batch Script (สร้างทีเดียว 5 คลิป)

```
สร้างบทพูดสำหรับคลิปสั้น 5 คลิป ธุรกิจ [ประเภท] ลงใน [TikTok/IG/YouTube]

แต่ละคลิปให้:
1. หัวข้อ + ประเภทคลิป (ให้ความรู้ / ขายของ / เล่าเรื่อง / รีวิว / สอนทำ)
2. บทพูดเต็ม (Hook + เนื้อหา + CTA)
3. Text overlay แนะนำ
4. ความยาวที่แนะนำ
5. เพลง/เสียงประกอบที่เหมาะ (แนะนำสไตล์)

ทั้ง 5 คลิปต้องหลากหลายรูปแบบ
```
